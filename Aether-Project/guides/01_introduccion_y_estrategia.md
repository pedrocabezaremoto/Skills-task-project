# Guía Introductoria y Estrategia Aether

**Naturaleza:** Proyecto Aether para Outlier enfocado en tareas visuales, cortas, creativas y descriptivas con límites estrictos de tiempo por tarea y un alto énfasis en creatividad y profesionalización de respuestas.

## 1. Requisitos para Habilitar Tareas
1. **Revisión de Términos:** Aceptación en Outlier Dash.
2. **Certificación Obligatoria:** Aether Intro Certification Course V2 (18 min).
3. **Acuerdos de Datos:** "NA Research Participation and Data Collection Agreement" por DocuSign.

Las tareas tipo `260107-solve-challenging-images` se desbloquearán si, además, se aprueba el 'code screening' y Aether se encuentra asignado como proyecto en la cola de tareas.

## 2. Gestión Operativa y Hubstaff
El proyecto se rige por AHT (*Average Handle Time*) predefinidos y no pueden ser superados si se espera pago eficiente. Hubstaff reinicia el contador todos los días a las 6:00 AM PST y está atado a un presupuesto colectivo para todos los colaboradores de Aether.

**Tiempos Ejemplares de AHT (Subproyectos críticos):**
- 251101-website-generation: 50 min.
- Astraea (Audio Annotation): 40 min.
- 250925-tripod-photos: 30.3 min.
- 251122-person-isolation-bbox: 29 min.
- Andromeda (Create Task): 22 min.
- 251216-pattern-extraction: 10.5 min.
- Aquila (AI Assistant Compare): 5 min.
- 250909-text-to-video-h2h: 3 min.
- 251030-audio-transcript: 2.5 min.

**Notas:**
- Si un AHT falta, es mejor reportar rápido el fallo y saltar la tarea para no penalizar tus estadísticas.
- **Límite Semanal Alcanzado:** Si Hubstaff dice esto, usualmente significa que el presupuesto DIARIO colectivo de todo el proyecto se agotó. Se añade presupuesto a las 6:00 AM PST.
- Las tareas `Dummy Task` de Outlier solo sirven de puente contable para cruzar horas con el registro de Hubstaff y tareas hechas en `multimango.com`

## 3. Estrategia y Excelencia ("Excelente" Rating)
1. **El Imperativo de Creatividad:** El cliente requiere obtener 'datos creativos de alta calidad'.
   - Un prompt seguro o funcional NO ES "Excelente".
   - Generar prompts detallados, altamente estilizados y basados en complejidad real.
2. **Cero Tolerancia a IA:** Queda PROHIBIDO usar ChatGPT para crear prompts, evaluar comparaciones o respuestas, o hacer copy-paste de formatos.
3. **Precisión Lingüística:** Utilizar Grammarly es obligatorio para garantizar cero errores gramaticales o de léxico.

**Ejemplos Específicos:**
- **251122 Person Isolation:** Aislar a TODAS las personas, incluso más de 10.
- **251216 Pattern Extraction:** Recorte perfecto sin distorsiones (ver Workflow 02).
- **251231 Object Relationship:** Descripciones de interacción lógica, complejas pero naturales "@man is wearing a @purple shirt".
