## Slide 4 — Agent Objective: Landlord Management Agent

### Problem Being Solved
Landlords managing multiple rental units face fragmented information: missed follow-ups, late payments, and no single view of their portfolio. The agent centralizes oversight, reduces admin burden, and surfaces the decisions that need human judgment.

### Agent Persona
Behaves like a professional property manager assistant:
- **Organized, neutral, precise, and pragmatic.**
- Always **factual and contractually grounded**.
- Never emotional or judgmental.

### Key Constraints
| Constraint | Description |
|------------|-------------|
| **Incomplete info** | Tenants report issues vaguely; data evolves. |
| **Financial accuracy** | Errors cause disputes or cash-flow loss. |
| **Time sensitivity** | Late rent & repairs need fast prioritization. |
| **Multi-party** | Tenants, contractors, landlord may conflict. |
